1. Software!!

------------

  1. Software!!

    2. We call software the set of programs, documentation and data closely related in order to make up a computer application. Software development requires the creation of a software project, and a group of people working in a coordinated way. Besides, software does not break along time, although its performance can be reduced because of its own updates and improvements.

  2. Somo Software components!!

    • Programs: sets of instructions that provide the desired functionality. They are written in a specific programming language.
    • Data: programs need data to work with. These data can be retrieved and/or stored from/in databases or files.
    • Documentation: software documents explain how to use it, which elements are part of it and how they are interconnected, so that it can be updated or fixed in the future.

  3. Software types

    • Application software: they bring some kind of service to the customer, such as text processors, spreadsheets, drawing
    • System software: they manage a computer system. They are basically programs that support or help other programs.

2. Languages and compilers

------------

The programing languages are used to provide instructions to a computer. After choosing a programming language we can create a source code.

2.1

There are two types of programming languages, a high level and low low level languages. High level languages are more understanble by the programmers, but the low level ones are closer to a machine code, meaningharder to understand.

Some examples of high level languages are: C, C++, C#, Java, Javascript, PHP, etc.

And the low level one is the assembly language

2.2

We have to use a compiler in order to translate the programing language into a machine code. There are some different language processors that can be used, depending on the language itself:

Compilers: create an executable file with the translation of the programing language into machine code Interpreters: translate from the specified programming language to machine code

Virtual machine: is an operating system or application environment that is installed on software, which imitates dedicated hardware. The end user has the same experience on a VM as they would on dedicated hardware.
